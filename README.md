# GLScoreMaker
Logiciel utilisé pour calculer le score lors de races Super Mario Maker

Raccourcis clavier:
- "Espace" pour ajouter une mort
- "s" pour ajouter un skip
- "l" pour ajouter un level

Vous pouvez changez le nombre de points gagnés/perdus pour les morts, skips et level réussis en selectionnant Configuration dans le menu en haut.
